# Automated E-commerce Flow using Selenium

This project demonstrates an end-to-end automation of a simple e-commerce workflow using the Selenium framework. The automation script is designed to interact with the [Sauce Demo](https://www.saucedemo.com/) website, including steps like logging in, adding an item to the cart, checking out, and making a dummy payment.

## Deliverables
- **Project Repository**: Contains the automation code using Selenium and a detailed README for execution steps.
- **Automation Framework**: The project follows a structured automation framework for seamless execution and reusability.
- **Video Demonstration**: Includes a recorded video showcasing the automated flows and the code in action.
- **Reusable Code**: The script is designed to be re-run on any new system with minimal configuration.

---

## Features Automated
1. **User Login**: Automated login with username and password.
2. **Item Selection**: Adding a specific item to the cart.
3. **Cart Management**: Viewing and proceeding with items in the cart.
4. **Checkout Process**: Filling out user details and simulating payment steps.
5. **OTP Verification**: Simulating an OTP verification process.
6. **Payment**: Completing a dummy transaction and verifying the success message.
